# **Лабораторная №3 — Экран авторизации (UIKit)**

### **Задача**

  
Реализовать экран авторизации: поля логина и пароля + кнопка входа.

Если логин и пароль верные — перейти на экран со списком фичей (можно заглушку: пустой экран/лейбл/print). 

  

### **Ключевые моменты**

- UI должен быть реализован с помощью **UIKit**. 
    
- Экран должен работать согласно выбранной архитектуре приложения (из Лабы 2). 
    
- Учесть маленькие экраны и клавиатуру, которая может перекрыть поля ввода. 
    

---

## **Требования (минимум)**

  
### **1) UI и верстка**

- Все элементы должны быть размещены через **Auto Layout Сonstraints**. 
    
- Экран должен корректно работать:
    
    - на маленьких экранах (например iPhone SE / mini)
        
    - при появлении клавиатуры поля не должны быть недоступны
        
    


  
### **2) Поведение и состояния**

- Кнопка “Войти” запускает проверку логина/пароля.
    
- При неверных данных показать ошибку (минимум: текст/алерт).
    
- Код должен быть “без крэшей”, с обработкой ошибок. 
    



### **3) Архитектура**

- Экран должен быть сделан в стиле архитектуры, выбранной в Лабе 2:
    
    - UI слой не хранит бизнес-логику авторизации
        
    - зависимости выражены через протоколы
        
    - навигация делается через роутер/координатор/аналог (как вы спроектировали)
        
    



### **4) Упрощение для этой лабы**


Авторизация может быть локальной (например: “правильная” пара логин/пароль захардкожена или лежит в конфиге). Сеть не требуется.

### **Подсказка (пример обработки клавиатуры через scroll)**

  
(Не обязательно делать именно так, но это рабочий путь)

```swift
final class LoginViewController: UIViewController {

    private let scrollView = UIScrollView()
    private let contentView = UIView()

    override func viewDidLoad() {
        super.viewDidLoad()
        setupLayout()
        setupKeyboardObservers()
    }

    private func setupKeyboardObservers() {
        NotificationCenter.default.addObserver(
            self,
            selector: #selector(keyboardWillChangeFrame(_:)),
            name: UIResponder.keyboardWillChangeFrameNotification,
            object: nil
        )
    }

    @objc private func keyboardWillChangeFrame(_ note: Notification) {
        guard
            let userInfo = note.userInfo,
            let endFrame = userInfo[UIResponder.keyboardFrameEndUserInfoKey] as? CGRect
        else { return }

        let keyboardInView = view.convert(endFrame, from: nil)
        let intersection = view.bounds.intersection(keyboardInView)
        scrollView.contentInset.bottom = intersection.height
        scrollView.verticalScrollIndicatorInsets.bottom = intersection.height
    }
}
```
